# Contribution Guidelines

## Adding a New Bank

- Fork this repository.
- Edit `banks.json` and add your bank entry:
  ```json
  {
    "name": {
      "en": "Bank Name",
      "ar": "اسم البنك"
    },
    "yearlyFee": "Yearly fee (in USD)",
    "perRequestFee": "Fee per request (in USD)"
  }
  ```
